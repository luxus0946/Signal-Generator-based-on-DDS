# 基于DDS的信号发生器

#### 介绍
本设计使用STC8H8K64U单片机通过IO端口控制波形发生器AD9833输出波形，经过低通滤波单元接入由MCP41010数字电位器控制的MS8051运放来将产生的信号幅值放大，并且配置了相应的按键输入和液晶屏显示波形的幅值、相位、频率等数据。
